# Jupyter_notebook

Docker image for Jupyter notebook, running on Ubuntu 18.04 and Python3.

As part of the the image, the following libraries are preloaded for consumption.

- numpy
- scikit-learn
- pyspark
- matplotlib
- nltk
- recordlinkage
